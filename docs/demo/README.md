# Demo Assets

Optional place for live-demo metadata or future recorded walkthrough assets. TRACaBot is already deployed as `@tracethembot`, and `docs/DEMO.md` is the repeatable reviewer/operator walkthrough.

Recommended files:

- `tracabot-live.md`: optional live deployment notes.
- `tracabot-demo-transcript.md`: optional transcript or chapter notes if a recording is added later.
- `tracabot-demo-link.md`: optional public YouTube/Loom/IPFS link if a recording is hosted externally later.

Do not commit secrets, `.env` files, Telegram bot tokens, private chat exports, or unredacted personal data.
