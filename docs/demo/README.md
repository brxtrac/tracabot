# Demo Assets

Recorded walkthrough assets for the OriginTrail DKG v10 Round 1 bounty submission. TRACaBot is already deployed as `@tracethembot`, and `docs/DEMO.md` is the repeatable reviewer/operator walkthrough.

Included files:

- `tracabot-demo.mp4`: recorded walkthrough satisfying the bounty requirement for a working demo.
- `tracabot-demo-cover.jpg`: cover image for the recorded walkthrough.

Optional files:

- `tracabot-live.md`: live deployment notes.
- `tracabot-demo-transcript.md`: transcript or chapter notes.
- `tracabot-demo-link.md`: public YouTube/Loom/IPFS link if the recording is hosted externally.

Do not commit secrets, `.env` files, Telegram bot tokens, private chat exports, or unredacted personal data.
