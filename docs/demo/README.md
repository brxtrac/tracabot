# Demo Assets

Deposit the recorded bounty demo assets in this directory before opening the `dkg-integrations` registry PR.

Recommended files:

- `tracabot-demo.mp4`: 5-8 minute walkthrough recording.
- `tracabot-demo-transcript.md`: optional transcript or chapter notes.
- `tracabot-demo-link.md`: public YouTube/Loom/IPFS link if the video is hosted externally.

Do not commit secrets, `.env` files, Telegram bot tokens, private chat exports, or unredacted personal data. If the video is too large for GitHub, commit only `tracabot-demo-link.md` with the hosted URL and keep this directory as the canonical place for demo metadata.
