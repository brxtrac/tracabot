# Contributing

tracabot is maintained for the OriginTrail DKG v10 Round 1 integration bounty with a minimum six-month support window after registry acceptance.

Before opening a PR:

1. Run `npm test`.
2. Keep Telegram, DKG, wallet, and OpenClaw secrets out of commits.
3. Describe any new network egress or DKG write operation in `SECURITY.md`.
4. Preserve the DKG v10 vocabulary: Context Graph, Working Memory, Shared Memory, Knowledge Asset, Knowledge Collection, Integration, Curator, SHARE, and PUBLISH.
